# 🏎️ Formula 1 Data Analysis with Python

## 📌 Overview

This project analyzes historical Formula 1 racing data using Python and Pandas. Multiple datasets are cleaned and merged to create a unified dataset for exploring driver performance, constructor dominance, race statistics, and circuit insights.

---

## 📂 Dataset

The project uses Formula 1 datasets including:

- Drivers
- Constructors
- Races
- Results
- Circuits
- Qualifying
- Pit Stops
- Seasons
- Driver Standings
- Constructor Standings
- Constructor Results

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Project Workflow

1. Load CSV datasets
2. Clean missing values
3. Merge multiple datasets
4. Perform Exploratory Data Analysis (EDA)
5. Visualize trends and statistics
6. Generate insights

---

## 📈 Analysis Performed

- Driver performance analysis
- Constructor performance
- Race winners
- Championship standings
- Circuit analysis
- Country-wise races
- Pole position analysis
- Pit stop statistics
- Season-wise trends

## 🚀 Installation

Clone the repository

```bash

```

Install dependencies

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

Run the notebook

```bash

```
